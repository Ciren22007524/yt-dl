## Enhancement 改動指南

如果要替這個專案加新功能或改進，以下是各類需求對應需要修改的檔案：

### 🔧 新增 API 端點（如：下載歷史、設定頁面）

| 改動檔案 | 做什麼 |
|----------|--------|
| `main.py` | 新增 `@app.get()` / `@app.post()` 路由 |
| `static/js/index.js` | 前端新增 `fetch()` 呼叫 |
| `templates/index.html` | 新增 UI 元素（按鈕、表格等） |

### 🎨 修改 UI / 新增頁面元素

| 改動檔案 | 做什麼 |
|----------|--------|
| `templates/index.html` | 修改 HTML 結構 |
| `static/css/index.css` | 新增 / 修改樣式 |
| `static/js/index.js` | 綁定新元素的事件邏輯 |

### 📡 修改即時通訊行為（進度條、通知）

| 改動檔案 | 做什麼 |
|----------|--------|
| `main.py` → `progress_hook()` | 修改後端推送的訊息格式 / 頻率 |
| `main.py` → `ConnectionManager` | 修改 WebSocket 連線管理邏輯 |
| `static/js/websocket.js` | 修改前端接收訊息後的 UI 更新邏輯 |

### 📥 修改下載行為（格式、品質、後處理）

| 改動檔案 | 做什麼 |
|----------|--------|
| `main.py` → `download_video()` | 修改 `ydl_opts` 設定（格式、品質、FFmpeg 參數） |
| `static/js/index.js` | 修改前端品質選項（`formatSelect.onchange`） |

### 📦 新增 Python 依賴套件

| 改動檔案 | 做什麼 |
|----------|--------|
| `pyproject.toml` | `poetry add <package>` 後自動更新 |
| `poetry.lock` | 自動更新（勿手動修改） |

## 關鍵技術決策筆記

1. **BackgroundTasks vs 多執行緒**：使用 FastAPI 內建的 `BackgroundTasks` 處理下載，簡潔但僅適合單一用戶場景。多人同時下載可能需要改用 Task Queue（如 Celery）。

2. **WebSocket 進度推送**：`progress_hook` 在同步的 yt-dlp 下載執行緒中運作，透過 `asyncio.run_coroutine_threadsafe()` 跨執行緒安全地將進度推送到 async WebSocket。

3. **loudnorm 音量標準化**：所有下載（包含 MP4）都會經過 FFmpeg 的 `loudnorm` 濾鏡處理，參數 `I=-16:TP=-1.5:LRA=11` 符合 EBU R128 廣播標準。

4. **imageio-ffmpeg**：不依賴系統安裝的 FFmpeg，而是透過 `imageio-ffmpeg` 套件自帶的 FFmpeg binary，確保跨平台一致性。

5. **PyInstaller 相容**：透過 `sys.frozen` 判斷是否為打包環境，動態切換 `BASE_DIR` 路徑，確保靜態檔案在 exe 中也能正確載入。
